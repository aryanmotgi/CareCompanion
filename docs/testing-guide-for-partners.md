## Step 2: Test the app

Go through each section and try everything:

- **Dashboard** — does it load? Does the data look right?
- **Medications** — view the list, try adding a new one, editing, deleting
- **Lab Results** — check if abnormal results are flagged
- **AI Chat** — send a message, see if you get a response
- **Settings** — toggle preferences, change theme
- **Notifications** — check the list, mark one as read

For the full detailed checklist, see [qa-checklist.md](./qa-checklist.md).

## Step 3: Report bugs

1. Click the **purple bug button** (bottom-right corner of the screen)
2. Describe what went wrong
3. Hit **Submit** — it automatically creates a ticket to fix

## If your account gets messed up

Go to **Settings** → scroll to **"Test Tools"** → tap **"Reset Test Data"** to restore everything to the starting state.
