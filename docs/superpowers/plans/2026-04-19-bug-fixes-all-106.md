Replace the big `Promise.all` block that uses `profile?.id || ''` with null-safe equivalents:

```ts
const profileId = profile?.id ?? null;

const [
  meds,
  docs,
  appts,
  labs,
  notifs,
  claimsData,
  priorAuthsData,
  fsaHsaData,
  memoriesData,
  conversationSummariesData,
  symptoms,
  [ins],
] = await Promise.all([
  profileId ? db.select().from(medications).where(eq(medications.careProfileId, profileId)) : Promise.resolve([]),
  profileId ? db.select().from(doctors).where(eq(doctors.careProfileId, profileId)) : Promise.resolve([]),
  profileId ? db.select().from(appointments).where(eq(appointments.careProfileId, profileId)) : Promise.resolve([]),
  db.select().from(labResults).where(eq(labResults.userId, dbUser!.id)).orderBy(desc(labResults.dateTaken)).limit(20),
  db.select().from(notifications).where(and(eq(notifications.userId, dbUser!.id), eq(notifications.isRead, false))).limit(10),
  db.select().from(claims).where(and(eq(claims.userId, dbUser!.id), eq(claims.status, 'denied'))).limit(5),

  loadConversationSummaries(dbUser!.id),
  db.select().from(symptomEntries).where(eq(symptomEntries.userId, dbUser!.id)).orderBy(desc(symptomEntries.date)).limit(14),
  db.select().from(insurance).where(eq(insurance.userId, dbUser!.id)).limit(1).then(rows => rows),
]);
```

Note: the `[ins]` destructure from the last `.limit(1)` query needs the array returned — keep `.then(rows => rows)` or adjust destructure to `insRows` then `const ins = insRows[0]`.

### Task 4: Fix onboarding "Go to Dashboard" 403 — send CSRF token in wizard fetches (Bug #4)

**Files:**
- Modify: `src/components/OnboardingWizard.tsx` — all `fetch('/api/records/...')` calls

**Root cause:** All `fetch` calls in the wizard POST to CSRF-protected endpoints but never include the `x-csrf-token` header. The cookie `cc-csrf-token` is set `httpOnly: false` so the client can read it.

- [ ] **Step 1: Add a helper at the top of OnboardingWizard.tsx**

```ts
function getCsrfToken(): string {
  return document.cookie
    .split('; ')
    .find(row => row.startsWith('cc-csrf-token='))
    ?.split('=')[1] ?? '';
}
```

- [ ] **Step 2: Add CSRF header to every fetch in the file**

Every `fetch('/api/records/...', { method: 'POST' | 'PATCH', headers: { 'Content-Type': 'application/json' }, ... })` call needs:

```ts
headers: {
  'Content-Type': 'application/json',
  'x-csrf-token': getCsrfToken(),
},
```

Apply to all fetch calls: `saveAndFinish`, `addMedication`, `addDoctor`, `addAppointment`, the inline profile saves on Steps 0–5, and the skip handler.

- [ ] **Step 3: Commit**

```bash
git add src/components/OnboardingWizard.tsx
git commit -m "fix(onboarding): send CSRF token header on all API fetch calls"
```
