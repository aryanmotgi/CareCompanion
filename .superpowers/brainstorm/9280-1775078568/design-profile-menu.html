Summary of the full redesign

  • 8 pages → 4 tabs + 3 menu items — Simpler mental model
  • Sidebar → Bottom tab bar — Native mobile pattern
  • Profile menu — Slides from right when you tap the avatar. Contains the less-used stuff.
  • Detail views use bottom sheets — Not new pages. Slide up, swipe down to dismiss.
  • Dark theme preserved — Same navy/slate/cyan palette, but cleaner cards without heavy glass effects.
