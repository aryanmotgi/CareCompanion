Key changes

  • Card-based feed — Each card = one thing that needs attention. Sorted by urgency.
  • Health score removed — It's a vanity metric that takes up prime space. Can live in profile if needed.
  • Quick prompts removed from home — Replaced by contextual "Quick Ask" card that suggests a relevant question.
  • No activity timeline — The feed IS the timeline, but each item is actionable.
  • Greeting + patient name — Personal, warm, immediately tells you whose care you're viewing.
