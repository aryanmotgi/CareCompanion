Care Tab — Segment Control

Toggle between Medications and Appointments via a sliding pill control. Meds grouped by "Refill Needed" → "Active". Appointments grouped by "This Week" → "Upcoming". Tap any card → bottom sheet detail overlay (slides up, swipe to dismiss). "+" button adds new items contextually.

Architecture

Each tab keeps its own route (/dashboard, /chat, /care, /scans) under the App Router. Profile/Connect/Settings render inside the ProfileMenu slide-out (client-side, no route change). Home feed aggregates data client-side from existing Supabase queries — no new API endpoints needed.
